# LinguaCore Design System

## 1. Visual Philosophy
**Theme Name**: Nebula Scholar
**Keywords**: Premium, Deep, Intelligent, Focus, Future.
**Aesthetic**: Glassmorphism elements, deep dark backgrounds, vibrant accent glows.

## 2. Color Palette

### Primary (Wisdom & Tech)
- **Primary**: `#7c3aed` (Violet-600) - For main actions, active states.
- **Primary-Hover**: `#6d28d9` (Violet-700).
- **Primary-Glow**: `rgba(124, 58, 237, 0.5)` - For shadows/glows.

### Secondary (Growth & Success)
- **Secondary**: `#10b981` (Emerald-500) - For success messages, progress completion.

### Accent (Insight)
- **Accent**: `#f59e0b` (Amber-500) - For highlights, stars, streaks.

### Background (Deep Space)
- **Bg-Dark**: `#0f172a` (Slate-900) - Main page background.
- **Bg-Darker**: `#020617` (Slate-950) - Contrast sections.

### Surface (Glass/Cards)
- **Surface**: `#1e293b` (Slate-800) - Card backgrounds.
- **Surface-Highlight**: `#334155` (Slate-700) - Hovers/Borders.

### Text
- **Text-Primary**: `#f8fafc` (Slate-50) - Headings.
- **Text-Secondary**: `#94a3b8` (Slate-400) - Body text.

## 3. Typography

### Font Families
- **Headings**: `Outfit` (Google Fonts) - Geometric, clean, modern sans-serif. Use for H1-H3.
- **Body**: `Inter` (Google Fonts) - Standard, highly legible. Use for paragraphs and UI text.
- **Fallbacks**: `sans-serif`.

### Scale
- **H1**: 3rem (48px), Bold (`700`).
- **H2**: 2.25rem (36px), SemiBold (`600`).
- **Body**: 1rem (16px), Regular (`400`).

## 4. Logo & Iconography
- **Symbol**: A geometric 'L' combined with a core node shape.
- **Style**: Gradient fill (Primary to Accent), sleek curves.

## 5. UI Components
- **Buttons**: Rounded-xl, slight shadow, hover lift effect.
- **Cards**: Dark slate background with thin 1px border (`border-slate-700/50`).
- **Inputs**: Dark background (`slate-900`), ring on focus (`ring-violet-500`).
